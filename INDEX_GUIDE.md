# A 股大盘指数预测指南

## 📊 支持的指数

| 代码 | 名称 | 说明 |
|------|------|------|
| 000001 | 上证指数 | 上海证券交易所综合指数 |
| 399001 | 深证成指 | 深圳证券交易所成份指数 |
| 399006 | 创业板指 | 创业板综合指数 |
| 399300 | 沪深300 | 沪深300指数 |
| 399905 | 中证500 | 中证500指数 |
| 000016 | 上证50 | 上海50指数 |
| 000905 | 中证500(主) | 中证500主指数 |

## 🚀 快速开始

### 预测上证指数
```bash
python predict_index.py --index 000001
```

### 预测沪深300
```bash
python predict_index.py --index 399300 --horizon 30
```

### 预测创业板指
```bash
python predict_index.py --index 399006 --horizon 60 --period 5y
```

## 📋 参数说明

| 参数 | 说明 | 默认值 | 示例 |
|------|------|--------|------|
| --index | 指数代码 | 必需 | 000001, 399001, 399006 |
| --period | 历史数据周期 | 3y | 1y, 2y, 3y, 5y |
| --horizon | 预测天数 | 30 | 10, 30, 60, 90 |
| --context | 上下文长度 | 512 | 256, 512, 1024 |
| --output | 图表保存路径 | 自动生成 | forecast.png |

## 📈 与个股预测的区别

| 特性 | 个股预测 | 指数预测 |
|------|----------|----------|
| 波动性 | 高 | 低 |
| 数据源 | 股票行情 | 指数行情 |
| 复权 | 前复权 | 不复权 |
| 预测难度 | 较高 | 相对较低 |
| 适用性 | 短期交易 | 趋势判断 |

## 💡 使用建议

1. **趋势判断**: 指数预测更适合判断市场整体趋势
2. **配置参考**: 可作为仓位配置的参考指标
3. **风险管理**: 结合个股分析，降低系统性风险

## ⚠️ 注意事项

- 指数预测反映市场整体预期
- 突发事件可能导致预测失效
- 建议结合基本面分析
